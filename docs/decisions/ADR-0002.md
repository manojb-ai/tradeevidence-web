# ADR-0002: Support Light, Dark, and System Themes

## Decision
TradeEvidence will support Light, Dark, and System themes, with Dark Mode as the default experience for the trading application.

## Context
Users will work in different environments and with different visual preferences. The product should support both active dashboard work and lighter reading contexts.

## Alternatives
- dark-only experience
- light-only experience
- single fixed theme

## Reason
Theme flexibility improves usability and aligns with the product's long-term design goals.

## Consequences
- broader accessibility and comfort
- more design system complexity
- stronger requirement for consistent theming

## Status
Proposed
