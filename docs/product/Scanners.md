# Scanners

## Purpose
Scanners help users discover ideas that match defined trading criteria.

## Users
- traders screening opportunities
- users exploring new setups

## Features
- configurable scan criteria
- result lists and summaries
- connection to watchlists and scoring

## Future ideas
- saved scans
- alert-based scan triggers
- historical scan review

## Dependencies
- scoring engine
- market data
- data model

## Open Questions
- Which scanner criteria should be supported first?
- How should scan results be ranked or grouped?
