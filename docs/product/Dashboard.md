# Dashboard

## Purpose
The dashboard is the primary workspace for traders to review evidence, context, and analysis at a glance.

## Users
- active traders
- users reviewing daily market context

## Features
- overview cards
- market summary context
- watchlist and score surfaces
- journal and alert highlights

## Future ideas
- configurable widgets
- saved layouts
- cross-module insights

## Dependencies
- scoring engine
- market data context
- design system

## Open Questions
- Which modules should appear on the default dashboard view?
- How much personalization should be supported at launch?
