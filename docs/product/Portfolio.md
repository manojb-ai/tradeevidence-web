# Portfolio

## Purpose
The portfolio module provides context around holdings, positions, and broader trading activity.

## Users
- traders reviewing active positions
- users comparing ongoing setups

## Features
- position overview
- portfolio context
- links to analysis and journaling

## Future ideas
- performance review views
- scenario analysis
- position-level notes and alerts

## Dependencies
- portfolio data model
- market data
- journal and reports

## Open Questions
- What portfolio details should be tracked at launch?
- Should portfolio views be separate from general research workflows?
