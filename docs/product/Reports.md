# Reports

## Purpose
Reports help users review evidence, scores, assumptions, and outcomes over time.

## Users
- traders reviewing past work
- users preparing internal summaries

## Features
- structured report views
- historical context
- score and evidence summaries

## Future ideas
- export-friendly formats
- daily or weekly review summaries
- AI-assisted report generation

## Dependencies
- journal data
- scoring engine
- AI services

## Open Questions
- What report templates should be supported first?
- Should reports be static or dynamic?
