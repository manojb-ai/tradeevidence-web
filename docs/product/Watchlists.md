# Watchlists

## Purpose
Watchlists help traders organize assets and ideas they want to monitor.

## Users
- traders tracking setups
- users comparing multiple opportunities

## Features
- saved watchlists
- status and context for each item
- quick access to scoring and notes

## Future ideas
- shared watchlists
- alert-based monitoring
- smart grouping by theme or sector

## Dependencies
- scoring engine
- market data
- data model

## Open Questions
- Should watchlists support private and shared modes?
- How should watchlist sorting be configured?
