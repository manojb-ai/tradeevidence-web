Evidence Score = Evidence quality

Application Layer = Workflow orchestration layer coordinating navigation, permissions, and request flows

Platform Services = Shared services supporting engines and workflows (market data, alerts, snapshots, auth, orchestration)
Data Layer = Persistence and historical record layer supporting reproducibility and validation
Provider Layer = External market-data provider integration layer behind internal service abstractions
Canonical Symbol Analytical Model = Shared symbol-based analytical profile consumed across dashboard, workspace, alerts, reports, and AI
Provider Abstraction = Architectural rule that business logic accesses market data only through internal services
Product Architecture = Conceptual product-level architecture and user-facing design direction defined in 03-Architecture.md
Engineering Architecture = Authoritative software implementation architecture defined in Master-System-Architecture.md
AI-DLC = AI-Driven Development Life Cycle method used selectively for building TradeEvidence
Engineering Lifecycle = TradeEvidence staged delivery workflow from framing through learning
Phase Gate = Explicit readiness checkpoint between lifecycle stages
AI-DLC Level = Risk-based rigor classification (Level 1 Lightweight, Level 2 Standard, Level 3 Controlled)
