# Canonical Terminology

Homepage = Authenticated landing page
Today's Briefing = Homepage hero
Decision Workspace = Primary research experience
Evidence Score = Evidence quality
Decision Confidence = Confidence in acting now
Decision Checklist = Preparation before acting
Decision Snapshot = Saved reasoning (future)
Decision Journal = Long-term learning system
AI-DLC = AI-Driven Development Life Cycle method used selectively for building TradeEvidence
Engineering Lifecycle = TradeEvidence staged delivery workflow from framing through learning
Phase Gate = Explicit readiness checkpoint between lifecycle stages
AI-DLC Level = Risk-based rigor classification (Level 1 Lightweight, Level 2 Standard, Level 3 Controlled)
ADR = Architecture Decision Record documenting an approved engineering decision
