# Design Tokens

## Purpose
Design tokens will provide a consistent foundation for spacing, color, typography, and visual treatment.

## Users
- designers
- UI engineers
- contributors implementing shared styles

## Features
- color tokens
- spacing scales
- typography scales
- elevation and state tokens

## Future ideas
- semantic tokens for score states and theme adaptation
- shared token documentation for designers and developers

## Dependencies
- design system
- implementation stack decisions

## Open Questions
- Which token categories should be defined first?
- How should light and dark themes share and differ in token values?
