# Frontend

## Purpose
The frontend provides the public website and the authenticated trading experience.

## Users
- visitors exploring the product
- traders using the application workspace

## Features
- marketing and product content
- dashboard and workspace surfaces
- research and analysis views
- theme support

## Future ideas
- reusable module shells
- richer data visualizations
- improved navigation patterns

## Dependencies
- Next.js
- React
- TypeScript
- Tailwind CSS
