## How to Use This Documentation
- Start with the product vision and philosophy documents for intent.
- Move to architecture, design system, and AI strategy for implementation framing.
- Use the decision workspace and trading profile documents as the primary product experience references.
- Keep new content concise, structured, and linked to the relevant source documents.
